# Requirements #

Go, obviously :)

Some gorilla packages:

```bash
$ go get github.com/gorilla/mux
```

Redigo:

```bash
$ go get github.com/garyburd/redigo/redis
```

# Build & Run #

```bash
$ go run *.go
```
